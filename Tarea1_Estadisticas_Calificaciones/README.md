# Calculadora de Estadisticas de Calificaciones

Proyecto escolar desarrollado en Java para procesar y analizar notas academicas utilizando arreglos unidimensionales.

---

## Descripcion General

El sistema permite capturar, validar y analizar un grupo de 8 calificaciones escolares de forma consecutiva. Genera automaticamente un reporte completo con metricas clave sobre el rendimiento general de los estudiantes.
